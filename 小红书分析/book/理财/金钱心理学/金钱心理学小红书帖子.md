# 💰 为什么你总是存不住钱？

---

🎬 想象这样一个场景：
双十一零点，你原本只想买一瓶洗面奶，结果凑单满减后买了一堆可有可无的东西。

月初还信用卡时你傻眼了：明明发了工资，怎么又不够花了？

这就是《金钱心理学》要告诉你的——存不住钱不是因为你不努力，而是你的大脑天生就不擅长处理财务问题。

❷【运气与风险】成功的人往往低估运气的作用 → 失败的人往往高估风险

❸【存钱是美德】致富不需要高收入，只需要不浪费 → 存款率决定财富速度

❷【自由才是财富】有钱的定义不是赚多少，而是你的选择权 → FIRE运动的核心

❺【市场先生的情绪】股价波动不能预测 → 你对波动的反应才是关键
