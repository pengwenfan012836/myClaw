## 下载方法

### 方法一：命令行下载

```bash
# 下载单个图片
curl -L -o 雷军.jpg "https://upload.wikimedia.org/wikipedia/zh/5/53/%E9%9B%B7%E5%86%9B_Lei_Jun.jpg"

# 批量下载（将下方 URL 保存到 urls.txt）
curl -L -o 雷军.jpg "https://upload.wikimedia.org/wikipedia/zh/5/53/%E9%9B%B7%E5%86%9B_Lei_Jun.jpg"
curl -L -o 雷军_2024-09-13.jpg "https://upload.wikimedia.org/wikipedia/zh/f/fc/%E9%9B%B7%E5%86%9B_2024-09-13.jpg"
curl -L -o 雷军_2019.png "https://upload.wikimedia.org/wikipedia/zh/c/c4/2019%E9%9B%B7%E8%BB%8D.png"
curl -L -o 雷军_20240425_1.jpg "https://upload.wikimedia.org/wikipedia/zh/c/ca/%E9%9B%B7%E5%86%9B_Lei_Jun_20240425_1.jpg"
curl -L -o 雷军_20240425_2.jpg "https://upload.wikimedia.org/wikipedia/zh/7/76/%E9%9B%B7%E5%86%9B_Lei_Jun_20240425_2.jpg"
curl -L -o 两会_2025.png "https://upload.wikimedia.org/wikipedia/zh/c/c4/Lei_Jun_at_Two_Sessions_2025.png"
```
