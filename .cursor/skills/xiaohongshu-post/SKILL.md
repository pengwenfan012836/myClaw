# 小红书帖子生成器

## 工作流程

1. **获取书籍信息**：搜索豆瓣获取基本信息（书名、作者、评分、评价人数）
2. **分析现有内容**：如果有现有读书分享文档，提取核心观点、金句、推荐理由
3. **生成帖子**：按下方SOP模板排版，输出纯文本格式
4. **保存文件**：自动保存为 `book/书名/书名小红书帖子.md`
5. **直接交付**：同时输出纯文本内容供用户复制

### 排版规范

| 要素 | 规范 |
|------|------|
| **emoji数量** | 全篇10-15个，每段开头使用 |
| **段落长度** | 每段2-3行，不超过50字 |
| **分隔符** | 用`---`分隔大段落 |
| **列表符号** | 用`🔸`或`▫️` |
| **数字标注** | 用`❶❷❸`或`①②③` |
| **重点强调** | 用`「」`引出核心概念，用`→`引出解释 |
| **标签数量** | 10-15个标签 |

### 高互动技巧

1. **开头Hook**：痛点共鸣 + 价值承诺
2. **悬念设置**：用"👇""往下看"引导继续阅读
3. **文末引导**：明确告诉读者"点赞""收藏""评论"
4. **标签组合**：热门标签（#读书推荐）+ 垂直标签（#书名）+ 场景标签

## 输出要求

### 文件保存
- 文件路径：`book/[书名]/[书名]小红书帖子.md`
- 格式：Markdown
- 文件名使用中文书名

### 格式要求
- **输出纯文本**，不是Markdown代码块
- **emoji正常显示**
- **用户可以直接复制粘贴**
- **不要使用代码块包裹输出内容**

### 内容要求
- 开头必须有Hook，吸引眼球
- 核心观点提炼到位，有洞察
- 金句选择有传播力的短句
- 标签要精准且有热度
- 整体风格温暖、治愈、有共鸣

### 标签参考

```
#读书推荐 #好书分享 #读书笔记 #书荒推荐
#小红书读书会 #读书博主 #阅读分享 #每日书单
#治愈系 #自我成长 #心理学入门 #干货分享
（+ 具体书名/作者标签）
```
