# 小红书帖子生成器

## 工作流程

1. **获取书籍信息**：搜索豆瓣获取基本信息（书名、作者、评分、评价人数）
2. **分析现有内容**：如果有现有读书分享文档，提取核心观点、金句、推荐理由
3. **搜索深度书评**：从豆瓣、知乎等平台搜索精彩书评和原文金句
4. **生成标题**：提炼书籍核心观点，生成一个吸引人的标题（必须！）
5. **随机选择模板**：从6种开头模板中随机选择1种
6. **生成帖子**：按模板排版生成内容
7. **保存文件**：使用Write工具保存为 `book/书名/书名小红书帖子.md`
8. **直接交付**：同时输出纯文本内容供用户复制

---

## 标题生成要求（必须执行）

每个帖子都必须生成一个**吸引人的标题**，放在正文开头之前。

### 标题原则

| 原则 | 说明 |
|------|------|
| **核心洞察** | 标题必须体现书籍最核心的观点或主题 |
| **有共鸣** | 能引发读者共鸣，让人想点进去看 |
| **简洁有力** | 一般8-20字，不要太长 |
| **有记忆点** | 读完能记住，容易传播 |

### 标题参考示例

| 书名 | 标题 |
|------|------|
| 《小狗钱钱》 | 复利是世界第八大奇迹 |
| 《小王子》 | 愿你出走半生，归来仍是少年 |
| 《蛤蟆先生去看心理医生》 | 能帮你的人是你自己，也只有你自己 |
| 《穷查理宝典》 | 我只想知道我会死在哪里 |
| 《被讨厌的勇气》 | 所谓的自由，就是被人讨厌 |
| 《原则》 | 痛苦+反思=进步 |
| 《思考快与慢》 | 你远没有自己以为的那么理性 |
| 《非暴力沟通》 | 会说话的人，一开口就赢了 |

### 标题生成方法

1. 从搜索到的书评和金句中提炼核心观点
2. 找到书中最有洞察力的一句话（或概括）
3. 用简洁有力的方式表达出来
4. 可以用数字、对比、反问等手法增加吸引力

### 标题输出格式

```
【标题】标题内容

---

[正文内容...]
```

## 排版规范

| 要素 | 规范 |
|------|------|
| **emoji数量** | 全篇10-15个，每段开头使用 |
| **段落长度** | 每段2-3行，不超过50字 |
| **分隔符** | 用`---`分隔大段落 |
| **列表符号** | 用`🔸`或`▫️` |
| **数字标注** | 用`❶❷❸`或`①②③` |
| **重点强调** | 用`「」`引出核心概念，用`→`引出解释 |
| **标签数量** | 10-15个标签 |
