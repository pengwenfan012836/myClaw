# 读书分享内容生成器

## 工作流程

1. **搜索书籍信息**：使用 WebSearch 搜索书名 + "豆瓣" 获取基本信息
2. **获取书评素材**：搜索热门书评和高分书评作为素材
3. **生成标题**：从书评和金句中提炼出书中最戳人的一句话作为标题（不超过20字）
4. **下载封面图片**：从豆瓣或Wikipedia下载书籍封面图，保存到书籍目录
5. **生成内容**：按下方模板输出 markdown 内容

---

## 图片获取方法

### 豆瓣封面图片URL格式
```
https://img1.doubanio.com/view/subject/l/public/s[编号].jpg
```
可以从豆瓣书籍页面获取实际URL，然后使用curl下载：
```bash
curl -L -o "[书名]封面.jpg" "[豆瓣封面URL]"
```

### Wikipedia公开图片（备选）
很多经典书籍在Wikipedia有公开图片，可搜索获取。

## 标题生成要求

### 标题来源
- 从搜索到的金句和书评中提炼
- 选择书中**最有洞察力、最戳人心**的一句话
- 也可以是对书籍核心观点的精准概括

### 标题规范
- **不超过20个字**
- 有共鸣、有传播力
- 能引发读者好奇心

### 标题示例

- 《小狗钱钱》 — 复利是世界第八大奇迹
- 《穷爸爸富爸爸》 — 穷人为钱工作，富人让钱为自己工作
- 《被讨厌的勇气》 — 所谓的自由，就是被人讨厌
- 《原则》 — 痛苦+反思=进步
- 《非暴力沟通》 — 会说话的人，一开口就赢了

## 内容生成要求

### 短文案规范
- 总字数：100-200字
- 风格：温暖、治愈、有共鸣
- 适合：小红书、抖音、朋友圈
- 包含：emoji 表情、话题标签

### 核心观点规范
- 3-5条，每条有标题 + 解释
- 避免书评原文摘要，重在提炼
- 观点要有洞察，不能只是复述情节

### 推荐理由规范
- 按人群分类：年龄、职业、场景、状态
- 每类人群2-3个即可
- 给出具体的阅读场景

### 金句摘录规范
- 优先选择有传播力的短句
- 每条不超过50字
- 可以是原文也可以是概括

---

## 注意事项

- **必须生成标题**：每本书都要提炼一个不超过20字的金句作为标题
- 先搜索再生成，不要凭空捏造信息
- 评分、人数等信息以搜索结果为准
- 生成的文案要符合中文社交媒体风格
- 保持内容的情感温度，不要太干巴巴
