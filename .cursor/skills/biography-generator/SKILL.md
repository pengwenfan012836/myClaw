# 传记资料生成器

## 工作流程

1. **搜索基本信息**：使用 WebSearch 搜索人物生平、重要事件、经典语录
2. **搜索演讲与访谈**：搜索该人物的公开演讲、访谈、纪录片资源
3. **创建目录结构**：在 `Biography/[人名]/` 下创建标准目录
4. **生成内容文件**：按下方模板生成所有 markdown 文件
5. **建立索引系统**：创建可跳转的索引文件

---

## 目录结构模板

```
Biography/
└── [人名]/
    ├── [人名]传记.md              ← 完整生平传记
    ├── [人名]语录.md              ← 经典语录集锦
    ├── [人名]资源.md              ← 推荐书单、纪录片、在线资源
    ├── [人名]演讲访谈索引.md      ← 总览列表（可跳转）
    └── 演讲与访谈/                ← 演讲与访谈详情子目录
        ├── [年份]-[日期]-[标题].md
        ├── [年份]-[日期]-[标题].md
        └── ...
```

## 内容生成要求

### 基本信息规范
- 姓名、出生日期、逝世日期、国籍等以权威来源为准
- 主要成就精简为3-5条核心贡献
- 使用表格形式便于阅读

### 时间线规范
- 按人生阶段划分章节（如：早年、创业、巅峰等）
- 重要事件标注年份，重大成就单独突出
- 由远及近排列

### 语录规范
- 优先选择有传播力的经典语录
- 每条语录标注出处（演讲、采访、书等）
- 按主题分类：人生、工作、创新、失败、死亡等

### 演讲访谈规范
- 精选15-20个最重要的演讲/访谈
- 每篇包含：基本信息 + 内容概述 + 经典语录 + 延伸阅读
- 提供原始视频/音频链接
- 斯坦福演讲等重要内容提供完整原文

### 资源推荐规范
- 书籍按传记类、管理类等分类
- 纪录片和电影标注年份和类型
- 提供可靠的在线资源链接

---

## 注意事项

- 先搜索再生成，确保信息准确
- 使用中文输出，内容符合中文阅读习惯
- 链接使用可靠来源（维基百科、官方网站等）
- 保持内容的完整性和深度
- 每个文件末尾添加"返回索引"链接
- 演讲访谈详情文件放在"演讲与访谈"子目录中
