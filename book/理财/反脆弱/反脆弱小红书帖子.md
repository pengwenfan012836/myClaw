❌ 大多数人都以为：脆弱就是弱，反之亦然

✅ 但其实：脆弱的真正反面不是强韧，而是"反脆弱"

这本书告诉我——

玻璃杯掉地上碎了，它是脆弱的
钢杯掉地上没事，它是强韧的
而病毒被抗生素攻击后，存活的那批反而产生抗药性，越挫越强——这叫反脆弱

真正的反脆弱，是能从混乱、波动、压力、不确定中获益并茁壮成长

🌟我从书里提炼了5个核心观点：

❶【脆弱三角】
脆弱类 ← 讨厌波动 ← 预测错误的代价随规模非线性增加
强韧类 ← 不在意波动 ← 结构稳定，不受影响
反脆弱类 → 喜欢波动 → 压力/混乱/错误反而让它进化

❷【火鸡问题】
"没有证据显示有风险" ≠ "有证据显示没有风险"
我们习惯用"目前没出事"来安慰自己，但黑天鹅事件从来无法预测

❸【槓铃策略】
两极化配置：90%极度安全 + 10%极高风险
→ 损失最多10%，收益却可能无限大
→ 上班族可以把90%时间花稳定工作，10%尝试高风险副业

❹【否定法】
知道什么错比知道什么对更有价值
→ 避开破产比寻找飙股更重要
→ 巴菲特：我只想知道我会死在哪里

❺【林迪效应】
能经历数次崩盘依然屹立的事物，比新兴独角兽更具反脆弱性
→ 时间越长存在，越可能继续存在
→ 选书原则：流传十年的书将再流传十年
